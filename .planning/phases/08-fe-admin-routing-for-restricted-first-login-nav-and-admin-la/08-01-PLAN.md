|----------|-------------|
| authenticated user → FE router | Users can type arbitrary `/admin/...` URLs directly into the browser |
| auth session → shell selector | Shell selection must not rely on mutable nav visibility alone |
| dropdown navigation → admin shell | The Administrator shortcut must not leak admin entry to non-admin roles |

| Threat ID | Category | Component | Disposition | Mitigation Plan |

| T-08-02 | T | shell selection in `Sidebar` / `MainLayout` | mitigate | Pass shell-specific nav contracts from routing instead of deriving shell from mixed pathname conditionals |
| T-08-03 | I | hidden code-graph admin route | mitigate | Include `/admin/code-graph/:kbId` in the route-permission resolver so permission checks do not silently fall through |
| T-08-04 | R | removed legacy admin routes | accept | Old non-admin admin URLs intentionally disappear per D-06/D-07; regression tests in 08-04 prove they no longer exist |
</threat_model>
