         - `tenant_id` varchar(255) not null
         - `created_by` varchar(255) nullable
         - `created_at` timestamp default now()

       - `queryMemoryMessagesSchema`: z.object with page (number optional default 1), page_size (number optional default 20), keyword (string optional), message_type (number optional)

    4. Create `extraction.prompts.ts` porting RAGFlow's PromptAssembler:
       - Export `SEMANTIC_EXTRACTION_PROMPT` (system + user template): "Extract factual statements, definitions, and key concepts from the conversation..."
       - Export `EPISODIC_EXTRACTION_PROMPT`: "Extract notable events, experiences, and interactions..."
       - Export `PROCEDURAL_EXTRACTION_PROMPT`: "Extract step-by-step procedures, workflows, and how-to instructions..."
       - Export `RAW_EXTRACTION_PROMPT`: simply returns the raw message content
       - Export `MEMORY_RANK_PROMPT`: "Given the query and memory items, rank them by relevance..."
       - Each prompt is a `{ system: string, user: string }` object with `{{conversation}}` placeholder

    5. Create barrel `index.ts` exporting MemoryModel, Memory type, schemas

    6. Register in ModelFactory:
       - Import `MemoryModel` from `@/modules/memory/models/memory.model.js`
