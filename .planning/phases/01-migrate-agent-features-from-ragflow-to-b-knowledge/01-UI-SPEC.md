## Spacing Scale

Declared values (must be multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps, inline padding, node port spacing |
| sm | 8px | Compact element spacing, node internal padding |
| md | 16px | Default element spacing, form field gaps, panel padding |
| lg | 24px | Section padding, canvas toolbar padding |
| xl | 32px | Layout gaps, panel-to-canvas gap |
| 2xl | 48px | Major section breaks, empty state vertical spacing |
| 3xl | 64px | Page-level spacing |

Exceptions:
- Canvas node minimum touch target: 44px (accessibility for drag handles)
- ReactFlow minimap: 120px x 80px minimum
- Debug panel fixed width: 360px (right-side sheet)

## Color

| Role | Light Value | Dark Value | Usage |
|------|-------------|------------|-------|
| Dominant (60%) | `hsl(0 0% 100%)` / `#ffffff` | `hsl(222 47% 11%)` / `#0f172a` | Background, canvas surface |
| Secondary (30%) | `hsl(210 40% 96.1%)` / `#f1f5f9` | `hsl(217 33% 17%)` / `#1e293b` | Cards, sidebar, node backgrounds, panel surfaces |
| Accent (10%) | `#0D26CF` | `#6380f0` | See reserved list below |
| Destructive | `hsl(0 100% 50%)` / `#ff0000` | `hsl(0 63% 45%)` / `#b91c1c` | Delete agent, delete node, error states |

**Accent reserved for:**
- Primary CTA buttons ("Create Agent", "Save Agent", "Run")
- Active sidebar navigation item (Agents)
- Selected canvas node border highlight
- Active tab indicator in agent config panels
- Debug "Step" and "Continue" action buttons
- Template "Use Template" button

**Canvas node category colors (secondary semantic, not accent):**

| Category | Light | Dark | Nodes |
|----------|-------|------|-------|
| Input/Output | `#3b82f6` (blue-500) | `#60a5fa` | Begin, Answer, Message |
| LLM/AI | `#8b5cf6` (violet-500) | `#a78bfa` | Generate, Categorize, Rewrite |
| Retrieval | `#10b981` (emerald-500) | `#34d399` | Retrieval, Wikipedia, Tavily |
| Logic/Flow | `#f59e0b` (amber-500) | `#fbbf24` | Switch, Condition, Loop, Merge |
| Code/Tool | `#ec4899` (pink-500) | `#f472b6` | Code, GitHub, SQL, API |
| Data | `#06b6d4` (cyan-500) | `#22d3ee` | Template, KeywordExtract, Baidu |

These colors are used as a 4px left-border or subtle background tint on canvas nodes only. They do not replace the accent color.

## Copywriting Contract

| Element | Copy |
|---------|------|
| Primary CTA | "Create Agent" |
| Empty state heading | "No agents yet" |
| Empty state body | "Build your first AI agent workflow. Start from scratch or pick a template." |
| Empty state action | "Create Agent" (primary) / "Browse Templates" (secondary) |
| Error state (load fail) | "Could not load agents. Check your connection and try again." |
| Error state (run fail) | "Agent run failed at node \"{node_name}\": {error_message}. Check the debug panel for details." |
| Destructive: delete agent | "Delete Agent": "This will permanently delete \"{agent_name}\" and all its run history. This cannot be undone." Confirmation: type agent name to confirm. |
| Destructive: delete node | No confirmation dialog. Undo via Ctrl+Z (canvas history). |
| Destructive: delete version | "Delete Version": "This will permanently delete version {version_label}. The current published version is not affected." |
| Save success | Toast: "Agent saved" |
| Run started | Toast: "Agent run started" |
| Debug step | Status badge on node: "Running..." / "Completed" / "Failed" / "Skipped" |
| Template clone | "Use Template": creates a copy with name "{template_name} (copy)" |
| Webhook URL display | "Webhook URL" label + copy button + "POST requests to this URL will trigger the agent." |

## Interaction Patterns

### Canvas Interactions

| Interaction | Behavior |
|-------------|----------|
| Add node | Drag from palette OR right-click canvas > "Add Node" OR Cmd+K palette |
| Connect nodes | Drag from output port to input port. Snap at 24px proximity. |
| Select node | Click node. Multi-select: Shift+click or drag-select rectangle. |
| Move node | Drag node. Snap to 8px grid. |
| Delete node | Select + Backspace/Delete. No confirmation (undo available). |
| Pan canvas | Click+drag on empty canvas OR middle-mouse-button drag |
| Zoom canvas | Scroll wheel. Pinch on trackpad. Zoom range: 25% to 200%. |
| Undo/Redo | Ctrl+Z / Ctrl+Shift+Z. Canvas state history (nodes, edges, positions). |
| Open config | Double-click node OR select node + Enter. Opens right-side sheet. |
| Canvas context menu | Right-click on empty canvas: Add Node, Paste, Select All, Fit View |
| Node context menu | Right-click on node: Configure, Duplicate, Delete, Copy |

### Agent Lifecycle

| State | Visual Indicator |
|-------|-----------------|
| Draft | Badge: "Draft" (muted) |
| Published | Badge: "Published" (accent/primary) |
| Running | Badge: "Running" (blue, animated pulse) |
| Failed | Badge: "Failed" (destructive) |
| Completed | Badge: "Completed" (emerald) |

### Debug Mode

| Element | Behavior |
|---------|----------|
| Toggle | Toolbar switch "Debug". Enables step-by-step execution. |
| Step button | Executes next node in topological order. |
| Continue button | Runs remaining nodes without pausing. |
| Node overlay | Shows execution status badge + duration on each node. |
| Input/Output inspection | Click node during/after debug to see JSON input/output in debug panel. |
| Breakpoint | Click node gutter dot to toggle breakpoint (pauses before execution). |

## Layout Contract

### Agent List Page (`/agents`)

```
+--sidebar--+--main-content-area-------------------+
|            | [heading] Agents          [+ Create] |
|  Agents *  | [tabs] All | My Agents | Templates  |
|  Chat      | [search bar]           [mode filter] |
|  Search    |                                      |
|  Datasets  | [card] [card] [card]                 |
|  Projects  | [card] [card] [card]                 |
|  ...       | [card] [card] [card]                 |
+------------+--------------------------------------+
```

- Card grid: 3 columns on desktop (>1280px), 2 columns (>768px), 1 column (mobile)
- Card content: agent name, mode badge, last modified, description truncated to 2 lines
- Card actions: kebab menu (Edit, Duplicate, Delete, Export JSON)

### Agent Canvas Page (`/agents/:id`)

```
+--toolbar-------------------------------------------+
| [< Back] [Agent Name] [Draft v] [Save] [Run] [Debug] [Share] [More v] |
+--canvas-area-----------------------+--config-panel-+
|                                    |               |
|   [node]--->[node]--->[node]       | Node Config   |
|      |                  |          | [form fields] |
|      v                  v          |               |
|   [node]            [node]         |               |
|                                    |               |
|  [minimap]  [zoom controls]        |               |
+------------------------------------+---------------+
```

- Canvas occupies remaining viewport (100vh - toolbar height 56px)
- Config panel: 360px fixed width, right side, collapsible via panel toggle
- Debug panel replaces config panel when debug mode is active (same 360px slot)
- Toolbar height: 56px, sticky top
- Minimap: bottom-left corner, 120x80px
- Zoom controls: bottom-left, above minimap

## Accessibility Contract

| Requirement | Implementation |
|-------------|---------------|
| Canvas keyboard navigation | Tab through nodes in DOM order. Arrow keys to move selected node. Enter to open config. |
| Screen reader node labels | Each node has `aria-label`: "{node_type}: {node_name}" |
| Focus visible | All interactive elements use `ring` token for focus outline |
| Color contrast | Node category colors used with text labels (not color-only semantics) |
| Reduced motion | Canvas animations respect `prefers-reduced-motion` (disable node transition animations) |
| Touch targets | All clickable elements minimum 44x44px (including node drag handles, port connection points) |
