**useAgentCanvas.ts** — Hook orchestrating canvas <-> server sync:
- Loads agent data via `useAgent(id)` on mount
- Converts AgentDSL nodes/edges to ReactFlow Node[]/Edge[] format and calls `canvasStore.loadFromDSL()`
- Provides `save()` function that: (1) converts ReactFlow nodes/edges back to AgentDSL, (2) strips UI-only fields (selected, dragging, measured) per RESEARCH anti-pattern, (3) calls `useUpdateAgent` mutation with the DSL, (4) calls `canvasStore.markClean()` on success, (5) shows toast "Agent saved"
- Provides `autoSave` interval (30 seconds) that saves if `isDirty` is true (per Pitfall 8)
- Provides `isLoading`, `isSaving`, `agent` from the underlying query/mutation
- Cleanup: clear autoSave interval on unmount
- JSDoc on hook and all returned values

**CanvasNode.tsx** — Generic node renderer for all operator types:
- Receives ReactFlow `NodeProps` with `data` containing: type (OperatorType), label, config
- Renders a card-like container with:
  - 4px left border in the node's category color (from NODE_CATEGORY_MAP + NODE_CATEGORY_COLORS)
  - Node type icon (map each OperatorType to a lucide-react icon)
  - Node title (14px semibold per UI-SPEC typography)
  - Node type label (12px regular, reduced opacity 0.7 per UI-SPEC)
  - Input port (Handle position=left, type=target) and output port (Handle position=right, type=source)
  - For switch/categorize nodes: multiple output handles with labels
- Min width 180px, min height 44px (touch target per UI-SPEC accessibility)
- aria-label: "{node_type}: {node_label}" per UI-SPEC accessibility contract
- Selected state: accent color border (#0D26CF light / #6380f0 dark)
- Dark mode support via Tailwind `dark:` classes
- JSDoc on component

**NodePalette.tsx** — Command palette for adding nodes (Cmd+K):
- Uses shadcn `Command` component (CommandDialog, CommandInput, CommandList, CommandGroup, CommandItem)
- Groups operators by category: Input/Output, LLM/AI, Retrieval, Logic/Flow, Code/Tool, Data
- Each item shows: category color dot, operator name, brief description
- Selecting an item: creates a new node at canvas center, calls canvasStore.addNode(), closes palette
- Keyboard: Cmd+K to open, Escape to close, arrow keys to navigate, Enter to select
- Search/filter by operator name
- JSDoc on component

**NodeConfigPanel.tsx** — Right-side configuration sheet:
- 360px fixed width per UI-SPEC, right side of canvas
- Shows when selectedNodeId is not null (from canvasStore)
- Header: node type icon + label, close button
- Body: dynamic form fields based on node type
  - For now, implement a generic JSON editor (textarea with JSON.stringify/parse) for node config
  - Individual operator forms will be added in later plans
- Footer: "Apply" button saves config to node via canvasStore.updateNodeData()
- Uses shadcn Sheet (side="right")
- JSDoc on component

**SmartEdge.tsx** — Custom edge with animation:
- Uses ReactFlow's `getBezierPath` for curved edges
- Animated dash pattern for "running" state (used during debug mode)
- Default: solid line with muted color
- Selected: accent color
- Label support for conditional edges (switch/categorize output labels)
- JSDoc on component
