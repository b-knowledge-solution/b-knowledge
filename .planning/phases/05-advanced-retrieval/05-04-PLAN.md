<!-- SSE Deep Research events (from Plan 02/03) -->
SSE data format for deep_research status events:
```typescript
{
  status: 'deep_research',
  subEvent: 'subquery_start' | 'subquery_result' | 'budget_warning' | 'budget_exhausted' | 'info',
  query?: string,      // the sub-query text
  depth?: number,      // recursion depth
  index?: number,      // sub-query index (1-based)
  total?: number,      // total sub-queries at this depth
  chunks?: number,     // chunks found for this sub-query
  message?: string,    // human-readable message
  tokensUsed?: number, tokensMax?: number,
  callsUsed?: number,  callsMax?: number,
  completed?: number,  // number of completed sub-queries (for budget_exhausted)
}
```
