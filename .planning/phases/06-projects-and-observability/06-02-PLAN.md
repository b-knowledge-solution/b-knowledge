1. Add to `ProjectsService`:

   **Member management (PROJ-03):**
   - `getProjectMembers(projectId: string)`: Query project_permissions WHERE project_id AND grantee_type='user', JOIN with users table to get name/email/role. Return array of { id, user_id, email, name, role, created_at }.
   - `addMember(projectId: string, userId: string, addedBy: string, tenantId: string)`: Insert project_permission row with grantee_type='user', grantee_id=userId, default tab permissions (tab_documents='view', tab_chat='view', tab_settings='none'). Check user exists in same tenant first. Audit log the action. Return the created permission.
   - `removeMember(projectId: string, userId: string, removedBy: string, tenantId: string)`: Delete project_permission WHERE project_id AND grantee_type='user' AND grantee_id=userId. Audit log the action. Reject if trying to remove the project creator.

   **Dataset binding (PROJ-02):**
   - `bindDatasets(projectId: string, datasetIds: string[], userId: string, tenantId: string)`: For each dataset_id, insert into project_datasets (skip existing). Audit log. Use single INSERT with ON CONFLICT DO NOTHING to avoid N+1 (Pitfall 3).
   - `unbindDataset(projectId: string, datasetId: string, userId: string, tenantId: string)`: Delete from project_datasets WHERE project_id AND dataset_id. Audit log. Immediate access revocation per user decision.

   **Cross-project dataset resolver (PROJ-04):**
   - `resolveProjectDatasets(userId: string, tenantId: string)`: Single query: `SELECT DISTINCT pd.dataset_id FROM project_datasets pd INNER JOIN project_permissions pp ON pd.project_id = pp.project_id WHERE pp.grantee_type = 'user' AND pp.grantee_id = ? AND pd.project_id IN (SELECT id FROM projects WHERE tenant_id = ?)`. Returns string[] of dataset IDs. Avoids N+1 (Pitfall 3).

   **Activity feed:**
   - `getProjectActivity(projectId: string, tenantId: string, limit: number, offset: number)`: Query audit_logs WHERE (resource_id = projectId OR resource_id IN (SELECT dataset_id FROM project_datasets WHERE project_id = projectId)) AND tenant_id = tenantId ORDER BY created_at DESC LIMIT/OFFSET. Returns paginated audit entries.

   - `activityQuerySchema`: { query: z.object({ limit: z.coerce.number().int().min(1).max(100).default(20), offset: z.coerce.number().int().min(0).default(0) }) }
