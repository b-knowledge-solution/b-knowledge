## Spacing Scale

Declared values (must be multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps, inline padding, badge internal padding |
| sm | 8px | Compact element spacing, table cell padding |
| md | 16px | Default element spacing, card internal padding |
| lg | 24px | Section padding, card content padding, stat card content padding (`p-6`) |
| xl | 32px | Layout gaps between major sections |
| 2xl | 48px | Major section breaks |
| 3xl | 64px | Page-level spacing |

Exceptions: Chart container height fixed at 300px matching existing `ActivityTrendChart.tsx`.

Source: Existing dashboard components (`p-6`, `gap-4`, `mb-6` patterns in `AdminDashboardPage.tsx`)

- Font weight 500 (medium) — used by existing shadcn Badge component internals, not directly specified in phase layouts

Source: Existing `StatCards.tsx` (`text-3xl font-bold`), `ActivityTrendChart.tsx` (`text-base font-semibold`), `AdminDashboardPage.tsx` (`text-lg font-semibold`)

## Interaction Contract

### Dashboard Analytics

| Interaction | Behavior |
|-------------|----------|
| Tab switch (Activity / Query Analytics / RAG Quality) | Instant content swap via `Tabs` component. Each tab has independent TanStack Query cache. No URL state needed |
| Date range change | Triggers TanStack Query refetch with new date params. Loading overlay appears during fetch |
| Refresh button | Icon-only button with `aria-label="Refresh data"` and tooltip "Refresh data". Calls `queryClient.invalidateQueries` for current tab's query keys. Uses `RefreshCw` Lucide icon that spins (`animate-spin`) while queries are fetching |
| "View in Langfuse" link | Opens new browser tab to `{langfuseBaseUrl}/trace/{traceId}`. External link icon next to text |
| Stale indicator | Show "Last updated: X minutes ago" below date range picker. Use `formatDistanceToNow` from date-fns |

### Project Management

| Interaction | Behavior |
|-------------|----------|
| Create project | Dialog with name + description fields. Submit creates project and redirects to detail page |
| Delete project | Dropdown menu action. Confirmation dialog: "This will remove all dataset bindings and member access. This action cannot be undone." Requires typing project name to confirm |
| Add member | Dialog with user search/select. Only shows users in same org. Role displayed as read-only badge (org role inherited) |
| Remove member | Inline button on member row. Confirmation dialog: "Remove [name] from [project]? They will lose access to all project datasets." |
| Bind dataset | Dialog with multi-select of available org datasets (excluding already-bound). Checkbox list with dataset name and document count |
| Unbind dataset | Inline remove button on dataset row. Confirmation: "Unbinding will immediately revoke access for all project members." |
| Activity feed pagination | "Load More" button at bottom. Loads next 20 items. No infinite scroll |

### Loading States

| State | Visual |
|-------|--------|
| Initial page load | `Spinner` component centered in content area |
| Tab data loading | Semi-transparent overlay (`bg-background/50`) with `Spinner` over content (existing dashboard pattern) |
| Table loading | Skeleton rows (3 rows of shimmer blocks matching column widths) |
| Empty data | `EmptyState` component with contextual icon and copy |

### Error States

| State | Visual |
|-------|--------|
| API error on analytics | `EmptyState` with `AlertCircle` icon, error message, "Try Again" button |
| API error on project CRUD | Sonner toast with error description |
| Langfuse link unavailable | Show "View in Langfuse" as disabled text with tooltip "Langfuse trace not available" |

## Recharts Configuration Contract

### Area Charts (Queries Over Time, Feedback Trend)

| Property | Value |
|----------|-------|
| Chart type | `AreaChart` with `monotone` interpolation |
| Container | `ResponsiveContainer` width 100%, height 300px |
| Margin | `{ top: 8, right: 8, left: -12, bottom: 0 }` |
| Grid | `CartesianGrid` strokeDasharray `3 3`, stroke from `--chart-grid`, vertical `false` |
| X-axis | `fontSize: 11`, fill `#94a3b8`, no axis line, no tick line |
| Y-axis | `fontSize: 11`, fill `#94a3b8`, no axis line, no tick line, `allowDecimals: false` |
| Area fill | Linear gradient (top 30% opacity to bottom 2% opacity) using `--chart-primary` |
| Stroke width | 2px |
| Dots | `dot={false}`, `activeDot={{ r: 4, strokeWidth: 2 }}` |
| Tooltip | Custom tooltip matching `ActivityTrendChart.tsx` pattern: rounded-lg, border, shadow-lg, 12px text |
| Legend | `iconType: "circle"`, `iconSize: 8`, `fontSize: 12`, `paddingTop: 8` |

Note: Chart axis font size (11px) and stat card metric size (30px) / weight (700) are inherited from existing `ActivityTrendChart.tsx` and `StatCards.tsx` respectively. They are not part of this phase's typography scale.

### Color Resolution

## Accessibility Contract

| Requirement | Implementation |
|-------------|---------------|
| Chart accessibility | Recharts provides native SVG `role="img"` and aria-label on chart containers. Add `aria-label` to each `ResponsiveContainer` with chart description |
| Table accessibility | Use semantic `<table>` from shadcn Table component (already accessible) |
| Keyboard navigation | All interactive elements (buttons, tabs, links) must be keyboard accessible via shadcn defaults |
| Color contrast | Stat card text is white on gradient backgrounds (existing tested pattern). Body text uses `foreground` on `background` (AAA compliant) |
| External links | "View in Langfuse" opens in new tab -- include `rel="noopener noreferrer"` and visually indicate external link with `ExternalLink` icon from Lucide |
| Destructive actions | Delete project requires typing project name (not just click confirm). Remove member requires explicit confirmation dialog |
| Refresh button | Icon-only `Button` with `aria-label="Refresh data"`, includes tooltip via `Tooltip` component displaying "Refresh data" on hover/focus |
