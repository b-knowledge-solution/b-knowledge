A simple presentational component that renders inline in the chat area during Deep Research streaming. Per user decision: show current sub-query label, sub-query count (N of M), budget warning when approaching limit, budget exhausted message.

Props interface:
```typescript
interface DeepResearchProgressProps {
  events: DeepResearchEvent[]
  isActive: boolean
}
```

Implementation approach:
1. Early return `null` if `!isActive || events.length === 0`
2. Derive display state from events array:
   - `latestStart`: last event with `subEvent === 'subquery_start'` (for current query label + index/total)
   - `budgetEvent`: last event with `tokensMax` set (for budget display)
   - `warning`: last event with `subEvent === 'budget_warning'`
   - `exhausted`: last event with `subEvent === 'budget_exhausted'`
3. Render a compact card with:
   - Brain icon (from lucide-react) + "Deep Research" label
   - Current sub-query: `t('chat.deepResearch.researching', { query })` with `(index/total)` counter
   - Budget bar: tokens used/max and calls used/max as text
   - Warning message in `text-amber-600 dark:text-amber-400` if budget_warning received
   - Exhausted message in `text-red-600 dark:text-red-400` if budget_exhausted received
4. Style with Tailwind: `rounded-lg border bg-muted/50 p-3 mx-4 mb-2` with dark mode support
5. Use `useTranslation` for all display strings
